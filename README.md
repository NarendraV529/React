# Profile Project

A personal portfolio website built with React. Features a clean, dark aesthetic with smooth animations and multiple pages.

## Pages

- **Home** — Hero section, stats, and skills overview
- **About** — Personal bio, details, and experience timeline
- **Projects** — Filterable project showcase with 6 projects
- **Contact** — Contact form with info section

## Tech Stack

- React 18
- React Router v6
- CSS (custom, no frameworks)
- Google Fonts (Playfair Display + DM Sans)

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm start

# Build for production
npm run build
```

## Project Structure

```
src/
  components/
    Navbar.js / Navbar.css
  pages/
    Home.js / Home.css
    About.js / About.css
    Projects.js / Projects.css
    Contact.js / Contact.css
  App.js
  App.css
  index.js
```

## Deployment
